# Admin Panel Features

## Overview
The admin panel is a comprehensive dashboard for managing all aspects of the portfolio website. Access it by clicking the "Admin" button in the navigation or navigating to `#admin`.

## Key Features

### 1. Media Upload System
**Location:** Projects Manager

**Features:**
- Drag-and-drop file upload
- Support for images (JPG, PNG, GIF, SVG)
- Support for videos (MP4, WebM)
- Automatic logo detection
- Real-time preview grid
- File size display
- Individual file removal
- Multiple file upload (up to 10 files)

**How to Use:**
1. Navigate to Projects Manager
2. Click "Add Project" or edit existing project
3. Use the media uploader to drag files or click to browse
4. Preview uploaded media before saving
5. Remove unwanted files by hovering and clicking the X button

### 2. Message Response System
**Location:** Messages Viewer

**Features:**
- View all contact messages
- Filter by: All, Unread, Archived
- Mark messages as read automatically
- Reply directly to clients
- Track reply history
- Archive or delete messages
- Email integration

**How to Use:**
1. Navigate to Messages
2. Click on a message to view details
3. Click "Reply to Client" to open reply box
4. Type your response
5. Click "Send Reply" to send via email
6. Previous replies are shown in green boxes

### 3. Click Tracking Analytics
**Location:** Analytics Dashboard & Overview

**Features:**
- Total clicks metric
- Click-through rate (CTR) for projects
- Most clicked elements tracking
- Click activity timeline
- Real-time click monitoring
- Geographic click data
- Element-level click tracking

**Tracked Elements:**
- Project "View Live" buttons
- Project "Case Study" links
- Contact form submissions
- Download CV button
- Social media links
- Email contact button

**Metrics Displayed:**
- Total Clicks: Overall click count
- CTR per Project: Click-through rate percentage
- Top Clicked Elements: Most popular interactions
- Click Timeline: When clicks occurred
- Click Activity Feed: Real-time click events

### 4. Project Management with Media
**Location:** Projects Manager

**Features:**
- Create, Read, Update, Delete projects
- Upload project screenshots, videos, and logos
- Add project features and tech stack
- Set live URLs and case study links
- Visual media gallery
- Logo upload for branding
- Inline editing
- Project categorization

**Project Fields:**
- Title
- Category
- Description
- Features (multiple)
- Tech Stack (multiple)
- Live URL
- Case Study URL
- Logo URL
- Media Gallery (images & videos)

### 5. Complete Analytics Suite

**Metrics Tracked:**
- Total Page Views
- Unique Visitors
- Total Clicks (NEW)
- Countries
- Session Duration
- Traffic Sources
- Top Pages
- Top Projects with CTR
- Click Tracking by Element

**Visualizations:**
- Line charts for views and visitors
- Bar charts for traffic sources
- Progress bars for page popularity
- Project performance cards
- Click tracking grid

### 6. Skills Management
**Features:**
- Create skill categories
- Assign icons to categories
- Add/remove individual skills
- Organize by technology type
- Visual skill display

### 7. Experience Management
**Features:**
- Add work experience entries
- Track contributions
- Display impact metrics
- Timeline visualization
- Edit and delete entries

### 8. Settings & Preferences
**Features:**
- Profile information management
- Social media links
- Notification preferences
- Email alerts configuration
- Password management
- Contact information

## Navigation

### Main Routes:
- **Portfolio:** Main website view (default)

## Tips for Best Use

1. **Media Upload:**
   - Use high-quality images (recommended: 1920x1080)
   - Keep video files under 50MB for best performance
   - Use descriptive file names

2. **Message Management:**
   - Reply to messages within 24 hours
   - Use the archive feature for old conversations
   - Check unread messages daily

3. **Click Tracking:**
   - Review click data weekly
   - Identify low-performing elements
   - Optimize based on CTR metrics
   - Use click heatmap insights for design decisions

4. **Analytics:**
   - Check daily for traffic trends
   - Monitor top-performing content
   - Track click-through rates
   - Adjust strategy based on click data

5. **Projects:**
   - Add media to showcase work
   - Upload project logos for branding
   - Keep descriptions concise
   - Update regularly with new work
